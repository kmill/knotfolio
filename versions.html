Releases of KnotFolio

Here are links to different versions of KnotFolio.

2023-06-19 (current)
Adds checkbox to toggle whether or not to compute invariants and identify the knot, useful to get PD codes of large knots.
2022-10-20
Virtual knot identification for knots with at least one virtual crossing (use a virtual Reidemeister I move if necessary).
2022-10-16
Cabled Jones polynomials, virtual knots, and cabled arrow polynomials. (No virtual knot identification in this release.)
2021-04-22
Improved beauty of "beautify" tool.
2021-04-21
Fixed infinite loop in "beautify" tool.
2021-04-20
Fixed KnotTheory PD convention. Added "beautify" isotopy tool. Changed default image thresholding value to be better for knotinfo images. Sped up calculation of Conway potential. Made lazy loading system for knot data.
2021-02-13
Added Conway potential.
2019-09-15
First release.
